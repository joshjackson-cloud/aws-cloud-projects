# AWS Cloud Projects

This repository contains hands-on AWS cloud engineering projects built as part of my cloud engineering training.

## Technologies Used
- Amazon Web Services (AWS)
- EC2
- S3
- IAM
- VPC (Foundations)
- Linux (Amazon Linux)
- Git & GitHub

## Projects

### 1. Cloud Web Server Deployment (EC2)
- Launched and configured Amazon Linux EC2 instances
- Configured security groups for SSH (22) and HTTP (80)
- Installed and managed NGINX web server
- Deployed a live web application
- Managed instance lifecycle (start, stop, terminate)
- Documented setup and architecture

### 2. Object Storage & Static Website Hosting (S3) *(In Progress)*
- Creating S3 buckets for object storage
- Hosting a static website using S3
- Configuring bucket policies and public access
- Enabling versioning and lifecycle rules

## Purpose
These projects demonstrate foundational cloud engineering skills, hands-on AWS experience, and the ability to design, deploy, and document cloud infrastructure.
